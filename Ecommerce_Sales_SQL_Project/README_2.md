# 📊 Sales Data Analysis Dashboard

## 🚀 Overview

This project focuses on analyzing a sales dataset (15,000+ records) to extract meaningful insights and build an interactive dashboard for business decision-making.

## 🛠️ Tech Stack

* Python (Pandas, NumPy)
* Power BI
* Excel

## 📊 Key Objectives

* Clean and preprocess raw sales data
* Perform exploratory data analysis (EDA)
* Identify key trends and patterns
* Build an interactive dashboard for KPIs

## 🔍 Analysis Performed

* Data cleaning (missing values, duplicates, formatting)
* Revenue trend analysis
* Region-wise and product-wise performance
* Monthly and category-based insights

## 📈 Dashboard Features

* Total Revenue & Sales KPIs
* Top-performing products
* Region-wise sales distribution
* Time-based trend analysis

## 📊 Key Insights

* Identified top revenue-generating regions and products
* Found seasonal trends in sales performance
* Highlighted underperforming segments
