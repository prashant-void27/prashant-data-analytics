# E-Commerce Sales Analysis (SQL Project)

## Objective
Analyze sales data to generate business insights including revenue trends, top customers, and regional performance.

## Tools Used
- SQL (MySQL/PostgreSQL)
- Excel / CSV Dataset

## Key Insights
- Total Revenue Calculation
- Region-wise Sales Analysis
- Top 5 Customers
- Monthly Sales Trends

## How to Run
1. Import the dataset into your SQL database.
2. Run queries from queries.sql file.
3. Analyze results and create dashboards if required.
