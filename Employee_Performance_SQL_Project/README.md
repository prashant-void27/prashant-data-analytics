# Employee Performance & Salary Analysis (SQL Project)

## Objective
Perform HR analytics to analyze salary distribution, department averages, and employee performance ratings.

## Tools Used
- SQL (MySQL/PostgreSQL)

## Key Insights
- Department-wise Average Salary
- Highest Paid Employee
- High Performing Employees
- Salary Ranking using Window Functions

## How to Run
1. Create employees and performance tables.
2. Insert sample data.
3. Run queries from queries.sql file.
